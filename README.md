# PHP OOP Learning Journey

This repository contains my journey of learning Object-Oriented Programming in PHP.

Instead of only studying the concepts, I practiced each topic by writing code and finally applied everything in a Mini E-Commerce project, which I later refactored to improve the code quality.

## Repository Structure

- **01-Classes-Basics** — Classes, Objects, Properties, and Methods.
- **02-Encapsulation&Constructor** — Encapsulation and Constructors.
- **03-Composition** — Composition between Classes.
- **04-Inheritance** — Inheritance and Code Reuse.
- **05-Abstraction** — Abstract Classes.
- **06-Polymorphism** — Polymorphism.
- **07-Interface** — Interfaces and Contracts.
- **Mini-E-Commerce** — Final project combining all previous concepts.

## Goal

The goal of this repository is to practice object-oriented programming in PHP and improve code quality through refactoring before moving on to Laravel.
