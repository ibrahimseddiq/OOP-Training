# Mini E-Commerce

This is a small PHP project I built while learning Object-Oriented Programming.

The idea was simple: after finishing the OOP topics, I wanted to apply everything in one project instead of keeping the concepts separate. After finishing the project, I went back and refactored the code to improve the class design and responsibilities.

## Features

- Product management
- User management
- Order creation
- Order items
- Order total calculation
- Promo code support
- Payment processing
- Preserving product prices at the time of purchase
- Input validation using Exceptions

## Concepts Used

- Encapsulation
- Composition
- Inheritance
- Abstraction
- Polymorphism
- Exception Handling
- Refactoring

## Notes

This isn't a complete e-commerce system. It's a learning project that helped me practice OOP by building something real before moving on to Laravel.
